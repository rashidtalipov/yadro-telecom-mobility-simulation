# Краткое описание проекта

Проект посвящен интеллектуальному управлению мобильностью в LTE/O-RAN-like сценарии. Основная задача - проверить, может ли ML-контроллер на основе RAN-телеметрии уменьшать нестабильные хэндоверы и ping-pong, сохраняя приемлемые QoS-метрики.

## Что реализовано

- многосотовый сценарий в `ns-3`;
- сбор RAN-телеметрии и событий хэндовера;
- подготовка датасетов для обучения;
- candidate-aware LSTM для trigger/target prediction;
- MLP baseline;
- replay-анализ политик;
- online closed-loop проверка через persistent Python inference worker;
- сравнение с A3 baseline.

## Почему это подходит для YADRO

Проект находится на стыке телекоммуникаций, алгоритмов, ML и системной разработки:

- есть предметная область сетей связи;
- есть C++/ns-3 часть;
- есть Python/PyTorch/data pipeline;
- есть метрики производительности и качества связи;
- есть baseline-сравнение и воспроизводимый экспериментальный подход.
