# Publication and Privacy Notes

This repository is intended as a public code and experiment-summary package for internship and technical review.

## Safe to Publish

The following types of materials are appropriate for this repository:

- original source code written for the project;
- compact result tables that do not reveal private data;
- short method descriptions;
- reproducibility instructions;
- newly written diagrams in Markdown/Mermaid;
- small model checkpoints only if they contain no private data and are intended for review.

## Do Not Publish Here

Do not upload:

- final accepted paper PDF;
- screenshots or crops of article pages;
- conference account exports;
- signed expert conclusions;
- contracts and payment documents;
- documents with stamps, signatures, phone numbers, or personal data;
- conference e-mails and reviewer correspondence;
- raw traces, databases, and full simulation output folders.

## About Figures From the Paper

For the GitHub repository, prefer diagrams recreated directly in Markdown/Mermaid or generated from public-safe CSV summaries.

Avoid using screenshots from the accepted paper before the conference/publisher workflow is complete. Even if the figures are your own, uploading final article pages or exact article figures before publication can create unnecessary ambiguity around exclusivity, version control, and publisher requirements.

## Recommended Wording in Applications

It is safe to write:

```text
The related paper has been accepted to IEEE EDM 2026, UID 6217. The repository contains a compact public code package and result summaries; the final paper PDF and private publication documents are not included.
```
